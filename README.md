My personal portfolio — a single-page site built with Next.js, statically exported.

## Develop

```bash
npm install
npm run dev      # http://localhost:3000
```

## Build

```bash
npm run build    # static site exported to ./out
```

## Test

```bash
npm run test:e2e
```

## Content

Site copy lives in `src/lib/site.ts` and projects in `src/lib/projects.ts`.
